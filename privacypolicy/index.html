Privacy Policy

Last updated:30/05/2025

The Shipment Scraper extension is designed to help users extract shipment information from the website https://solutions.inet-logistics.com and save it locally as a text file.

Data Collection

This extension does not collect, store, transmit, or share any personal data, user activity, or usage analytics. All data processing occurs locally in your browser.

Permissions

The extension requests the following permissions:

  • "downloads" – Required to save the extracted data as a text file.
  • "host_permissions" for https://solutions.inet-logistics.com/* – Required to access and read data from the shipment page.

Third-Party Access

This extension does not communicate with any external servers or third-party services.

Changes to This Policy

This privacy policy may be updated to reflect changes in the extension's functionality. If changes are made, the "30/05/2025" date at the top of the page will be revised.
